GIT steps for changes

  1. git branch [name]
  2. git checkout [branch]
  3. git push --set-upstream origin [branch]
  4. git add [first file]
  5. repeat above for each new file (if applicable)
  6. git commit -m 'Message'
  7. git checkout main
  8. git merge main [branch]
  9. git push
